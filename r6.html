⛔ COSA FA R6 - SMART GRID

Funzione Critica:

R6 è un BLOCCO DI SICUREZZA per la PDC (Pompa Di Calore) tramite SMART GRID

💡 COSA È LO SMART GRID?

Smart Grid è un sistema di gestione intelligente dell'energia elettrica che:

  • Monitora i consumi energetici in tempo reale
  • Ottimizza l'uso dell'energia quando costa meno
  • Può bloccare temporaneamente dispositivi ad alto consumo (come la PDC)
  • Comunica con il gestore elettrico per evitare sovraccarichi
  • R6 = interruttore Smart Grid che può fermare la PDC per risparmiare energia

🔌 LOGICA OPERATIVA

🟢 Sequenza Normale (PDC Funzionante):

  1. LED R6 SPENTO (armadio Shelly)
  2. Quadratino R6 NON ROSSO (QE1)
  3. Controller YOUTAKY richiede calore

🔴 Sequenza Blocco (PDC Bloccata):

  1. LED R6 ACCESO (armadio Shelly)
  2. Quadratino R6 ROSSO (QE1)
  3. Segnale BLOCCO su visore YOUTAKY
  4. PDC (Pompa Di Calore) BLOCCATA tramite SMART GRID

  2. Se R6 sta bloccando: Procedura reset 2 minuti
  3. Verificare sblocco su YOUTAKY

  5. Se non funziona → chiamare MARCO (335-1354309)

Problema: Blocco Smart Grid continuo
